## How to use it

```
1. Go to ascenly.org
2. Get Tampermonkey (works on Chrome and Firefox)
3. Paste the Ascenly script
4. Open Edgenuity and you'll see the Ascenly UI
```

> If you run into anything, the [Discord](https://discord.gg/edgenuity) is the fastest way to get help.

## It covers everything

**Answer types:** Multiple choice, matching, drag and drop, fill in the blank, dropdown, short answer, constructed response

**Written work:** Essays, journals, and paragraphs are generated automatically by the AI writer

**Other stuff:** Videos get skipped, activities auto-advance, and locked assignments can be unlocked

Works on the original Edgenuity and the newer **EdgeEX** version.
